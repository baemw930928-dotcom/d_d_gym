# DDGYM Class Management Tool

## Build Instructions
This project uses Vite, React (TypeScript), and Material UI.

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## Troubleshooting
If you encounter TypeScript errors during build, ensure you are using the latest code from the `main` branch. 
The project has been updated to be compatible with MUI v6+ (Standard Grid replaces Grid2, `slotProps` for TextField).
